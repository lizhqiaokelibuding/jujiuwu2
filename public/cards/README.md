# 卡牌分类说明

卡牌已按以下 6 类分类，每张卡仅属于一个类别：

## 蔬菜类 (vegetables/)
白萝卜、牛蒡、莲藕、毛豆、甜菜、紫苏叶、蘑菇、山药、大葱、小麦、大米、面粉、迷迭香

## 水果类 (fruits/)
杏、葡萄、柠檬

## 种子类 (seeds/)
白萝卜种子、牛蒡种子、莲藕种子、毛豆种子、甜菜种子、紫苏叶种子、蘑菇种子、山药种子、杏种子、柠檬种子、葡萄种子、大葱种子、迷迭香种子

## 建筑类 (buildings/)
水田、池塘、麦田、磨坊、酿造桶

## 菜品类 (dishes/)
奶油蘑菇炖菜、山药泥、山药麦饭、杏仁豆腐、炖土豆、炸鸡块、烤串拼盘、烤饭团、牛排、玉子烧、刺身拼盘、筑前煮、盐、糖、酱油、酱汁、红酒

## 动物类 (animals/)
三文鱼、秋刀鱼、甜虾、小牛、小鸡
